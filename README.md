#What does Doing Ending Leaving mean?

##Doing

* It auto starts the video
* Makes it go full screen (you can change that part).

##Ending

*Turns of fullscreen

##Leaving

*It Deletes the video from existance.

#How to use it?

* You can use for a intro to a website,
* A popup ad,
* And other reasons!
